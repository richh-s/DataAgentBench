You are working with two datasets to solve the query: "Which English-language books in the 'Literature & Fiction' category have a perfect average rating of 5? List all of them please."

- This dataset contains review information from Amazon on Books caregory (ratings, text, helpfulness votes, etc.) up to 2023.
- Fields:
    - rating (float)           
    - title (str)            
    - text (str)               
    - parent_asin (str)       
    - review_time (str)

2. **books_dataset** {books_info.sql}
- This dataset contains books information from Amazon (descriptions, price, details, title. etc.) up to 2023.

    - subtitle (str)        
    - author (str)         
    - rating_number (int)   
    - features (str)       
    - description (str)    
    - price (float)        
    - store (str)          
    - categories (str)      
    - details (str)
